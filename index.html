업로드한 오디오 파일이나 유튜브 링크를 바탕으로 연습용 리드 시트를 만들어줍니다. 멜로디 노트, 기타 코드, 간단 오선보, 구간 요약까지 한 번에 확인할 수 있습니다.

모바일 대응 유튜브 링크 지원 PDF / 이미지 저장

사용 흐름

  1. 오디오 파일을 올리거나 유튜브 링크를 붙여넣기
  2. 브라우저에서 멜로디와 코드 흐름 분석
  3. 리드 시트, 오선보, 구간 요약 확인 후 저장

입력

파일 업로드와 유튜브 링크 분석을 모두 지원합니다.

파일을 올리거나 유튜브 링크를 입력하면 분석을 시작합니다.
